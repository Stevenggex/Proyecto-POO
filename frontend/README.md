# Frontend - Tienda de Videojuegos (Vista Web)

Sitio estático en HTML, CSS y JavaScript que muestra el catálogo de videojuegos registrado desde la app de escritorio (backend). Es solo de lectura: no permite crear, editar ni eliminar nada desde aquí.

## Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)

## Estructura (pendiente de desarrollar)

- `index.html` → página principal con el catálogo
- `styles.css` → estilos del sitio
- `app.js` → lógica de renderizado del catálogo
- `data/` → datos exportados desde el backend (JavaFX)
- `img/` → imágenes de los videojuegos exportadas desde el backend

## Cómo correrlo localmente

**Opción A (recomendada):** Usa la extensión "Live Server" en VS Code, clic derecho en `index.html` → "Open with Live Server".

**Opción B:** Abre `index.html` directamente en el navegador.
